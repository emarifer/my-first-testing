# my-first-testing

### App with VueJS that shows various testing examples.

## Project setup
```
npm install
```

### Start unit testing in watch and coverage mode
```
npm t
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your unit tests
```
npm run test:unit
```

### Run your end-to-end tests
```
npm run test:e2e
```
